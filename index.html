Hey there! I'm April Denise Dizon, a recent Computer Science grad from Rutgers University. I'm a software enginner and an indie video game developer based in the tri-state area.

In my previous years, I've participated in research in AI and pathfinding at Rutgers University, under the Aresty Research Program. Now, I'm using those skills to design apps and software as a solo developer, with my current project being a bullet hell game in Unity. Because I currently work alone, I've learned to wear many hats - studying outer disciplinary skills like graphic design and marketing.

As a software engineer I'm interested in AI and backend, and as a video game developer, I'm interested in programming. My favorite part of a project is the planning and conceptualization. Here, I get to visualize how different systems can efficiently work well together, and still allow room for newer systems in the future. A well thought out system will not only run well for the user but will be easy to understand by the programmers. Clean and robust code is itself art!

A fun fact: I'm also a dancer! I do ballet, ballroom, and swing! I've tried figure skating as well, but I'm still learning.
